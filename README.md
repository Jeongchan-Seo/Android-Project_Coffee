# 커피에 대한 상식을 알고싶다면 ?
커피를 좋아하거나, 다양한 커피에 대해 알고싶은 사람들을 위한 커피 소개 앱입니다.

커피 이외에도 다양한 정보들을 제공합니다.

## 제공하는 정보
+ 커피들의 종류
+ 나라별 원두의 종류
+ 커피 머신의 종류
+ 프랜차이즈의 종류
